DataVerse AI (FINAL3) — Project Entry Point

This repo was reorganized so it’s easy to navigate:

1) Documentation (all Markdown is inside `docs/`)
   - docs/README.md  (project overview + architecture)
   - docs/INDEX.md   (index of all docs)
   - docs/STRUCTURE_GUIDE.md (where everything lives)

2) Source code
   - dataverse_backend/   (FastAPI backend)
   - dataverse_frontend/  (Next.js frontend)
   - retail-agent/        (optional legacy agent)

3) Scripts (run/start/deploy/testing helpers)
   - scripts/start-all.ps1   (Windows: start backend + frontend)
   - scripts/start-all.bat   (Windows wrapper for start-all.ps1)
   - scripts/START.bat       (Windows: backend + open dashboard HTML)
   - scripts/deploy.sh       (Linux/Mac: docker deployment helper)
   - scripts/deploy.ps1      (Windows: docker deployment helper)
   - python scripts/demo_client.py  (API demo client)

4) Sample datasets
   - data/sample_products.csv
   - data/sample_data.csv
   - data/retail_mart_processed_v1.csv

Quick run (Docker):
  - From the repo root (the folder containing docker-compose.yml):
    docker-compose up -d --build

Quick run (Windows local dev):
  - From the repo root:
    powershell -ExecutionPolicy Bypass -File scripts/start-all.ps1
